# Spring篇

Jimmer本身是高度中立的，不和任何容器类框架耦合，但可以轻松和任何容器类框架耦合。

由于SpringBoot极其常见，Jimmer内置了SpringBoot Starter，用于简化开发。

其中，革命性的功能为前后端免对接和远程关联。
