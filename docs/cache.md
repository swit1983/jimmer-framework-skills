# 缓存篇

Jimmer提供了强大的缓存支持能力。

* 是否使用缓存，对开发人员完全透明
* 使用哪些缓存技术，由开发人员决定
* 不仅支持简单的

  对象缓存，
  还支持属性级缓存，包括关联缓存和计算缓存。
  最终提供任意形状的数据结构的缓存能力，而非简单对象的缓存能力
* 以权限系统为代表的功能会让不同用户看到不同的数据，因此，Jimmer通过

  多视图缓存来让不同用户看到不同的缓存。
* 强大的缓存一致性支持，开发人员专注于修改数据库即可，Jimmer自动

  保证数据库和缓存的一致性
